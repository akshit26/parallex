# Parallex Cyberpunk Experience

A cinematic, scroll-driven digital agency website built with Next.js App Router, TailwindCSS, GSAP ScrollTrigger, and Framer Motion.

## Run locally

```bash
npm install
npm run dev
```

## Architecture

- `src/components/sections/*` scene-based modular sections.
- `src/hooks/*` reusable animation hooks.
- `src/data/content.ts` configurable services and case studies.
